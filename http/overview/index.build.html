Condensation is a complete data system. It can

  • store and retrieve data like a database or a file system,
  • send data to other users or devices like e-mail or messengers,
  • and share and synchronize data like cloud storage.

Condensation is a distributed system, and does not need any central server. Actors (users, or their devices) are communicating directly with each other, or through a server accessible to both of them.

As opposed to cloud storage, Condensation is a distributed protocol, and does not need any central server.

end-to-end encrypted

Condensation thereby follows a distributed actor-message-passing approach, and encrypts all data end-to-end.
