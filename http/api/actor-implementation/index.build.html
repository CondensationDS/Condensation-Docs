The Condensation library includes an actor implementation using a data tree suitable for most applications. This document discusses how to implement your own actor.

Private data

First and foremost, chose a way the actor structures and stores its own private data. You may use data trees, or your own forward-mergeable data structure.

Actor group

If your actor is supposed to operate as part of an actor group, think about how actors join a group, and how they share private data within the group.
